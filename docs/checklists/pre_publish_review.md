# Pre-Publish Review Checklist

> Complete this checklist before any Instagram post is published.
> If any required item is unchecked, publication is blocked.

## A) Post Metadata
- [ ] Campaign / initiative:
- [ ] Post type (Reel/Story/Feed/Carousel):
- [ ] Draft URL / asset location:
- [ ] Intended publish date/time:
- [ ] Owner:

## B) Policy & Platform Compliance
- [ ] Content complies with Instagram Community Guidelines and Terms.
- [ ] No prohibited content (hate, harassment, violence, explicit exploitation, self-harm encouragement).
- [ ] No deceptive/inauthentic behavior (spam, fake engagement, impersonation).
- [ ] Factual claims are sourced and verifiable.

## C) Brand Safety Filters
- [ ] Hate/abuse screen passed (automated + manual spot check).
- [ ] Medical/legal/financial claims reviewed for substantiation and required disclaimers.
- [ ] No guaranteed outcomes or misleading promises.
- [ ] Sensitive content escalated to Compliance/Legal when required.

## D) Rights & Asset Clearance
- [ ] All media (music, images, clips, templates) have valid usage rights.
- [ ] UGC permissions documented where applicable.
- [ ] Trademark/logo usage verified.
- [ ] Attribution included where required.

## E) Disclosure & Monetization
- [ ] Sponsored relationship disclosed clearly (`#ad` / paid partnership label).
- [ ] Affiliate relationship disclosed clearly where relevant.
- [ ] Monetized claims are truthful and substantiated.
- [ ] Disclosures are visible (not hidden in truncated text or hashtag blocks).

## F) Human Approval Gates
- [ ] Editorial approval: ____________________  Date: __________
- [ ] Compliance approval: __________________  Date: __________
- [ ] Legal approval (if needed): ____________ Date: __________
- [ ] Brand approval (if needed): ____________ Date: __________
- [ ] Phase requirement met (Pilot/Stabilization/Mature): __________

## G) Final QA
- [ ] Caption, tags, mentions, links, and CTA verified.
- [ ] Accessibility check complete (alt text/subtitles/readability as applicable).
- [ ] Geography/language targeting appropriate.
- [ ] Tracking parameters and analytics tags verified.

## H) Publish Decision
- [ ] **APPROVED TO PUBLISH**
- [ ] **HOLD / REVISE**
- [ ] **REJECTED**

## I) Post-Publish Monitoring (Required)
- [ ] 1-hour check completed.
- [ ] 24-hour performance and policy scan completed.
- [ ] Any incident logged and routed per governance policy.
- [ ] Underperformance action selected (iterate/repackage/retarget/retire).
